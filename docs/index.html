Atomic

Building blocks that come together to create flexible yet consistent UI design components and patterns.

Themeable

Customisation options that are easy to get started with and extend with a little CSS know-how.

Extendable

Customise by extending the object-oriented PHP classes, override HTML output using Blade, and more.

Abstract

Inspired by and compatible with many WordPress blocks, but generic enough to be used in any PHP project.

Semantic

Carefully crafted HTML output prioritising semantic meaning and accessibility.

Clean

BEM class naming and data attributes combine to balance scoped and global styling while avoiding class name soup.
