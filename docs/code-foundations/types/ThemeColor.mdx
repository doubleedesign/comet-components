# ThemeColor
Enum used to specify valid values for theme colour properties

## Basic usage
```php
use Doubleedesign\Comet\Core\ThemeColor;

$result = ThemeColor::PRIMARY;
$value = $result->value; // returns 'primary'
```

---
## Available Values
| Case        | Value       |
|-------------|-------------|
| `PRIMARY`   | `primary`   |
| `SECONDARY` | `secondary` |
| `ACCENT`    | `accent`    |
| `ERROR`     | `error`     |
| `SUCCESS`   | `success`   |
| `WARNING`   | `warning`   |
| `INFO`      | `info`      |
| `LIGHT`     | `light`     |
| `DARK`      | `dark`      |
| `WHITE`     | `white`     |
