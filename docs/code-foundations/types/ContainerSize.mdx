# ContainerSize
Enum used to specify valid values for container size properties

## Basic usage
```php
use Doubleedesign\Comet\Core\ContainerSize;

$result = ContainerSize::WIDE;
$value = $result->value; // returns 'wide'
```

---
## Available Values
| Case | Value |
|------|-------|
| `WIDE` | `wide`|
| `FULLWIDTH` | `fullwidth`|
| `NARROW` | `narrow`|
| `DEFAULT` | `default`|
