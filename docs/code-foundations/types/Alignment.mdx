# Alignment
Enum used to specify valid values for layout alignment properties

## Basic usage
```php
use Doubleedesign\Comet\Core\Alignment;

$result = Alignment::START;
$value = $result->value; // returns 'start'
```

---
## Available Values
| Case           | Value          |
|----------------|----------------|
| `START`        | `start`        |
| `END`          | `end`          |
| `CENTER`       | `center`       |
| `JUSTIFY`      | `justify`      |
| `MATCH_PARENT` | `match-parent` |

---
## Methods
### `fromString(string $value)`

Converts a string to the corresponding enum case.

#### Example usage
```php
use Doubleedesign\Comet\Core\Alignment;

$result = Alignment::fromString('left');
```

#### Supported Input Mappings
| Input       | Result               |
|-------------|----------------------|
| `'start'`   | `self::START`        |
| `'left'`    | `self::START`        |
| `'top'`     | `self::START`        |
| `'end'`     | `self::END`          |
| `'right'`   | `self::END`          |
| `'bottom'`  | `self::END`          |
| `'center'`  | `self::CENTER`       |
| `'justify'` | `self::JUSTIFY`      |
| `default`   | `self::MATCH_PARENT` |
