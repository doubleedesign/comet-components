# Orientation
Enum used to specify valid values for layout orientation properties

## Basic usage
```php
use Doubleedesign\Comet\Core\Orientation;

$result = Orientation::HORIZONTAL;
$value = $result->value; // returns 'horizontal'
```

---
## Available Values
| Case         | Value        |
|--------------|--------------|
| `HORIZONTAL` | `horizontal` |
| `VERTICAL`   | `vertical`   |
