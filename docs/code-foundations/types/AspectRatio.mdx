# AspectRatio
Enum used to specify valid values for aspect ratio properties

## Basic usage
```php
use Doubleedesign\Comet\Core\AspectRatio;

$result = AspectRatio::STANDARD;
$value = $result->value; // returns '4:3'
```

---
## Available Values
| Case | Value |
|------|-------|
| `STANDARD` | `4:3`|
| `PORTAIT` | `3:4`|
| `SQUARE` | `1:1`|
| `WIDE` | `16:9`|
| `TALL` | `9:16`|
| `CLASSIC` | `3:2`|
| `CLASSIC_PORTRAIT` | `2:3`|
